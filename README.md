# Project-Ascension

Pre-production workspace for a handcrafted Minecraft progression project built around primitive survival, authored tech eras, physical spaceflight, and planet-by-planet advancement.

## Repository Structure

- `docs/` contains the game design documentation.
- `design/` is reserved for diagrams, worksheets, and supporting design artifacts.
- `roadmap.md` tracks pre-production milestones.
- `todo.md` tracks unresolved design decisions and follow-up work.

## Current Focus

The current goal is to lock design constraints before implementation:

- progression grammar
- planet framework
- combat and boss unlock philosophy
- technical responsibility split between mods, scripting, and custom code
