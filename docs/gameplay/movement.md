# Movement

## Purpose

Defines how movement evolves across the project.

## Movement Principles

- Every era should expand player movement options.
- Planet identity should be partly expressed through traversal.
- Movement upgrades should create new routes, not just speed.

## Movement Categories

- Primitive traversal
- Vertical traversal
- Vehicle-assisted travel
- Low-gravity traversal
- EVA or orbital maneuvering
- Advanced personal mobility

## Open Questions

- Which movement verbs are core to the project?
- When does low gravity first appear?
- How much personal flight is acceptable before vehicles lose value?
