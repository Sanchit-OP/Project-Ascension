# Survival

## Purpose

Defines hunger, temperature, hazards, shelter pressure, and moment-to-moment survival identity.

## Survival Principles

- Early survival should be demanding but understandable.
- New planets should add distinct survival rules, not reskins.
- Survival prep should create meaningful logistics decisions.

## Hazard Categories

- Temperature
- Atmosphere
- Radiation
- Storms
- Terrain hazards
- Hostile fauna

## Open Questions

- How punishing should early-game survival be?
- Which hazards are global systems versus planet-specific systems?
- How much downtime is acceptable for prep loops?
