# Exploration

## Purpose

Defines discovery, navigation, world structure, and rewards for going off the critical path.

## Exploration Principles

- Exploration must return value.
- Information is a reward class, not only loot.
- Optional sites should deepen mastery or world understanding.

## Reward Types

- Unique resources
- Blueprints or schematics
- Lore fragments
- Shortcut activation
- Upgrade components
- Boss intel

## Open Questions

- How guided versus unguided should navigation be?
- Do players earn map data, scan data, or coordinates as progression items?
- How much optional exploration is required to avoid content drought?
