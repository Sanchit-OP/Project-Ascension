# Bosses

## Purpose

Defines boss progression, encounter roles, and unlock rewards.

## Boss Design Rules

- Every mandatory boss unlocks a new capability.
- Every boss tests preparation, not only damage output.
- Boss arenas must reinforce the planet identity.
- Boss loot must not obsolete the rest of the era.

## Encounter Template

- Progression role
- Required prep
- Core mechanics
- Failure conditions
- Unlock reward
- Optional repeat reward

## Open Questions

- Which existing mod bosses stay, change, or get replaced?
- How many mandatory bosses per era is sustainable?
- Should some planet completions use non-boss victory conditions?
